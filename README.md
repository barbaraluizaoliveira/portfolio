# Portfólio Next.js (App Router)

Currículo/portfólio com integração a APIs públicas (ZenQuotes e GitHub).

## Rodar localmente
```bash
npm i
npm run dev
```
Abra http://localhost:3000

## Publicação
- Faça fork/clone, suba para um repositório seu e conecte na Vercel.
- Configure o projeto como `Next.js` (nenhuma variável obrigatória).

## Páginas
- `/` Home: mini bio e frase do dia (ZenQuotes API)
- `/sobre`: tecnologias/módulos usados
- `/academico`: timeline acadêmica
- `/profissional`: timeline profissional
- `/projetos`: repositórios do GitHub via API
